Coffee Builder Order (Best Flow)
1️⃣ Choose Drink Type

This determines the brewing method.

Drink Type	Base Method
Latte	espresso + milk
Cappuccino	espresso + milk foam
Americano	espresso + water
Drip Coffee	brewed coffee
Cold Brew	cold extraction
Flat White	espresso + microfoam
Cortado	espresso + equal milk

So if you want a latte, the customer selects:

➡ Latte (espresso based)

Important Design Tip

Let drink type come first, because it determines:

brewing method

ratios

equipment

Then everything else modifies it.

Final Structure for the App

1. Drink Type
2. Bean / Blend
3. Roast Level
4. Strength (shots)
5. Milk
6. Syrup / Sweetener
7. Add-ins
8. Toppings
